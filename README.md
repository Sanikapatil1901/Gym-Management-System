# 🏋️ Gym Management System

A console-based Gym Management System developed in Python using Object-Oriented Programming (OOP) and File Handling. This application helps manage gym members, their membership details, payment status, and reports efficiently.

---

## 📌 Project Overview

The Gym Management System is designed to simplify gym administration by allowing users to perform member management operations through a menu-driven console interface.

The project demonstrates the implementation of Python fundamentals, Object-Oriented Programming concepts, file handling, and modular programming.

---

## ✨ Features

- ➕ Add New Member
- 📋 View All Members
- 🔍 Search Member
- ❌ Delete Member
- 💳 Manage Membership Plans
- 💰 Track Payment Status
- 📄 Generate Reports
- 💾 Store Data Using Text Files
- 🖥️ Menu-Driven Interface

```bash

```

2. Open the project folder

```bash
cd Gym-Management-System
```

3. Run the project

```bash
python main.py
```
